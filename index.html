Overview

The Die

QuVaultX-PC — 185 mm² in TSMC N5 FinFET. Six stacked functional layers integrated over a UCIe 3.0 fabric at 64 GT/s connecting to the Ogopogo quad-core RISC-V manycore.

Core Architecture

Three Algebraic Kernels

Every cryptographic operation reduces to polynomial NTT, multi-scalar multiplication, or hash evaluation. One shared Algebra Core serves all five subsystems — no duplication, full concurrency.

Five Pillars

Full Stack in Silicon

ZKPE · FHES · MPCF · FEOE · CEU — the complete programmable cryptography stack on a single die. No other chiplet simultaneously targets all five layers of the hierarchy.

Security

Physical Root of Trust

The Key Lifecycle Vault provides CC EAL6+ isolation, hardware TRNG, PUF identity, and anti-DPA masking. Emergency zeroisation within 4 UCIe clock cycles.

Memory

Memory Bandwidth

Four HBM3 channels deliver 3.28 TB/s aggregate — fueling key material, witness data, ciphertext buffers, and MSM base tables simultaneously at full chip load.

Novel Architecture · CEU

The Cryptomaton

A program that knows its own secrets — opaque even to the die that runs it. FHE-encrypted registers, oblivious branching, and ZK proofs of correct execution baked into silicon.

I

Encrypted Program Store

Instructions fetched and decrypted inside the secure EPS. Plaintext instructions never appear on any external bus.

II

Opaque Private State

256-entry register file maintained in LWE-encrypted form. Arithmetic operates on ciphertext. Decryption key held exclusively in the KLV — never exported.

III

Oblivious Branching

Branch targets computed via blind rotation — no control-flow leakage against the host fabric. Which path was taken remains cryptographically hidden.

IV

Selective Decryptability

Output ciphertexts carry function-specific keys from the FEOE. Only the designated recipient with skf can decrypt f(y) — a silicon realisation of FE semantics.

QuVaultX-PC: A Programmable Cryptography Chiplet Architecture — From Lattices to Cryptomata, Silicon Foundations for Zero-Knowledge, Homomorphic Computation, and Obfuscation.

Kiprop Yego — Independent Research | Chiplet Security Systems | QuVaultX

Download White Paper PDF
Abstract

We present QuVaultX-PC, a five-pillar silicon architecture that accelerates the full hierarchy of modern cryptographic machinery: from NIST post-quantum standards to ZK proofs, FHE, MPC, functional encryption, and the novel Cryptomaton Execution Unit — a microarchitecture for running non-interactive, privacy-preserving autonomous programs directly on silicon.
